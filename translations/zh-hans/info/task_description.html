第一款台式可编程计算器诞生于 20 世纪 60 年代中期。其中包括 Mathatronics Mathatron（1964 年）和 Olivetti Programma 101（1965 年末），它们都是固态、台式、打印、浮点、代数输入、可编程、存储程序的电子计算器。这两款计算器均可由最终用户编程并打印出结果。Programma 101 的销售范围更广，并增加了通过磁卡离线存储程序的功能。Find more on wikipedia page...

在本系列任务中，您将制作一个初级计算器。

作为输入，您需要按下一串按键，作为函数的结果，您需要显示按下最后一个键时屏幕上会显示什么。注意，这并不总是表达式的结果。实际上，使用实物计算器或应用程序确实有助于你捕捉边缘情况。

在第六个任务中，添加了大写字母 "B"（退格）和大写字母 "C"（清除）按钮，前者用于删除前一个字符，后者用于删除所有输出，包括错误信息。

预期行为：

  • 开头的零应删除，只有零的数字 - 转换为单个零；
  • 数字之间的 +- 符号应取最后一个；
  • "==" 表示重复上次操作；
  • "+="或"-="--在组合前对数字（或运算结果）进行加减运算（将数字加倍或减去本身）；
  • 计算器忽略在第 5 位之后输入的数字；
  • 数字 < 0 的"-"不占位数；
  • 如果 abs(结果)大于 99999，结果显示为 "错误"；
  • 对于浮点数，如果 abs(结果) 的整数部分大于 9999，则结果显示为 "错误"；
  • 对于 float，如果数字的总长度超过 5 位数，则应四舍五入到 5 位数（1.23456 -> 1.235）；
  • 对于浮点数，应去掉开头和尾部的零（如果可能，去掉". "之前的零）： 0.1200 -> .12 , 123.00 -> 123..应在四舍五入后进行： 1.000123 -> 1. .只有在输入数字结束后（按下非数字字符），才能删除尾部的零；
  • 当 B 字符后没有符号且按下新的数字时 - 与前一个数字连接：12+B3 -> 123 .

输入：字符串。

输出： 字符串字符串。
